# Commander Token Table Aids

Private, print-light table aids for the Commander decks tracked in
`docs/20_commander_token_print_manifest.md`.

This folder is the editable source for the generated PDF:

```text
printables/commander-token-table-aids/
  index.html
  styles.css
  data.js
  render.js

output/printables/commander-token-table-aids.pdf
```

The cards are intentionally text-first. They do not use official card frames,
logos, or token art.

For the current PDF, skip the cover if desired and print pages 2-11 at 100%
scale, double-sided. The duplex pairs are 2/3, 4/5, 6/7, 8/9, and 10/11.
Cut along the borders and sleeve card-sized pieces in front of bulk cards.
