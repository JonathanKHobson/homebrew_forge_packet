# Figma MTG Card Assets Pack

This pack points at Kyle's local SVG export from the Figma community MTG card designer:

`/Users/kyle/Documents/My Games/Magic The Gathering/Magic The Gathering Card Assets`

It is the current primary pack for modern MTG-style frames. GenevensiS remains useful as a fallback and as a mana-symbol source until a cleaner dedicated mana-symbol pack is wired.

Symbol sources used by this pack:

- mtg-vectors for set and watermark SVGs.
- Keyrune as the set-symbol font fallback/source reference.
- GenevensiS mana PNGs for mana/tap/action symbols in this first pass.

The Figma full-card SVGs include editor placeholder labels and sample mana icons. The manifest opts into render hints so Homebrew Forge covers those baked placeholders before drawing live card data.
