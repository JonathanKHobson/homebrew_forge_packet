# Glossary

## Asset pack

A local folder containing frame assets, symbols, fonts, and layout maps, described by a manifest.

## Asset source

A configured origin that can import or sync assets into an asset pack, such as a local directory, npm package, or GitHub release.

## Card record

A row/object representing a physical card's identity: name, layout, collector number, status, print count, and metadata.

## Card face record

A row/object representing one printable face/panel: mana cost, type line, rules text, art, P/T, loyalty, etc.

## Debug frame

A plain non-production frame used only for development/testing when no real asset pack is installed.

## Export profile

A configuration controlling output target, size, watermark, format, compression, filenames, and package shape.

## Reference cache

Local copy of Scryfall/MTGJSON data used for wording, import, and comparison.

## Source of truth

The CSV/YAML/JSON files and local assets in the set folder. Rendered images and XML are generated artifacts, not source of truth.
